# 타깃 기업 선정 기준표

## 1. 평가 목적

DX 컨설팅을 제안할 만한 건강기능식품 D2C 브랜드를 선별하기 위해 100점 만점의 평가 기준을 수립했습니다.

평가 기준은 아웃바운드 영업 관점에서 “컨설팅 니즈가 있을 가능성”과 “제안했을 때 설득력 있는 문제 가설을 만들 수 있는지”를 중심으로 설계했습니다.

## 2. 평가 기준

| 평가 기준 | 배점 | 판단 기준 |
| --- | ---: | --- |
| 판매 채널 복잡도 | 20 | 자사몰, 쿠팡, 스마트스토어, 올리브영, 카카오, 인스타그램 등 여러 채널을 운영하는가 |
| CRM/재구매 중요도 | 20 | 정기구독, 재구매, 멤버십, 리뷰 관리, 고객 세그먼트 관리가 중요한 비즈니스인가 |
| 데이터 활용 가능성 | 20 | 광고 성과, 구매 이력, 고객 행동, 상품별 매출 등 분석할 데이터가 많아 보이는가 |
| 성장/채용 신호 | 15 | 마케팅, CRM, 데이터 분석, 이커머스 운영 관련 채용이나 성장 뉴스가 있는가 |
| DX 제안 적합도 | 15 | 리포팅 자동화, CRM 자동화, 대시보드, 고객 분석 같은 컨설팅 과제를 제안하기 좋은가 |
| 접근 가능성 | 10 | 회사/브랜드 정보, 판매 채널, 담당 조직, 연락 가능성이 어느 정도 확인되는가 |
| 합계 | 100 | 70점 이상이면 우선 검토 대상, 80점 이상이면 제안 타깃 후보 |

## 3. 점수 해석

| 점수 | 의미 | 조치 |
| --- | --- | --- |
| 85점 이상 | 최우선 아웃바운드 타깃 | 상세 기업 분석 및 제안서 작성 |
| 75-84점 | 제안 가능성이 높은 타깃 | 추가 조사 후 우선순위 판단 |
| 65-74점 | 추가 조사 필요 | 채용, 채널, 성장 신호 보강 |
| 64점 이하 | 우선순위 낮음 | 현재 단계에서는 제외 |

## 4. 핵심 판단 질문

- 이 회사는 판매 채널이 많아서 성과 데이터가 흩어져 있을 가능성이 큰가?
- 재구매와 CRM이 매출에 중요한 구조인가?
- 광고비, 고객 데이터, 구매 데이터를 통합해서 보면 개선 여지가 큰가?
- 채용공고나 뉴스에서 데이터, CRM, 퍼포먼스 마케팅 니즈가 보이는가?
- DX 컨설팅 제안서를 보냈을 때 “우리 문제 맞다”라고 느낄 만한 포인트가 있는가?

## 5. 평가 방식

각 기업은 공개 정보 기반으로 6개 항목을 점수화했습니다. 점수는 내부 데이터가 아닌 외부 관찰 신호에 따른 사전 가설이며, 실제 영업 전에는 최신 채용공고와 공식 채널을 재확인해야 합니다.
