# 공개 데이터 기반 분석 리포트: 프롬바이오

## 1. Executive Summary

프롬바이오는 건강기능식품 제조 및 판매를 주력으로 하는 상장사입니다. 공개 자료를 기준으로 보면 2023년에는 매출 감소와 영업손실 전환이 발생했고, 이후 온라인 채널 확대와 OEM 수주 확대를 통해 수익성 개선을 추진하는 흐름이 확인됩니다.

본 리포트의 핵심 결론은 다음과 같습니다.

- 프롬바이오의 DX 과제는 단순한 자사몰 개선이 아니라 `채널 구조 전환과 수익성 관리`입니다.
- 공개 자료에서 온라인 채널 비중 확대와 자사몰/앱 운영 신호가 확인되므로, 온라인 CRM과 채널별 손익 관리가 중요합니다.
- OEM 수주 확대는 생산 가동률과 원가율 개선으로 연결될 수 있어, 제조 운영 KPI까지 포함한 대시보드가 필요합니다.
- 다만 고객 수, 광고비, 채널별 마진, 앱 활성 지표는 공개되지 않았으므로 정량 ROI 계산에는 내부 데이터가 필요합니다.

## 4. 실적 관점 분석

2023년 프롬바이오는 연결 기준 매출 667억 원, 영업손실 154억 원을 기록했습니다. 공개 보도에서는 전년 대비 매출이 33% 감소했고, 상장 이후 첫 적자 전환이라고 설명합니다.

이 수치는 두 가지 문제를 시사합니다.

1. 매출 회복만큼이나 비용 구조 개선이 중요합니다.
2. 채널별 매출과 비용을 함께 보는 관리 체계가 필요합니다.

홈쇼핑 판매수수료와 마케팅 비용 증가가 실적 부진 원인으로 언급되었기 때문에, 단순히 매출액만 보는 대시보드로는 충분하지 않습니다. 채널별 판매수수료, 광고비, 할인비, 물류비를 함께 보아야 실제 수익성 개선 여부를 판단할 수 있습니다.

## 5. 채널 전환 관점 분석

공개 보도에 따르면 프롬바이오의 자사몰 등 온라인 채널 매출 비중은 2023년 32.8%에서 2024년 2분기 약 50% 이상으로 증가했습니다.

이는 긍정적인 전환 신호이지만, 온라인 비중 확대가 곧바로 수익성 개선을 의미하지는 않습니다. 온라인 채널 역시 광고비, 할인비, 플랫폼 수수료, 물류비가 발생하기 때문입니다.

따라서 핵심 질문은 다음과 같습니다.

| 질문 | 필요한 분석 |
| --- | --- |
| 온라인 매출 비중 확대가 영업손익 개선으로 이어지고 있는가? | 월별 채널별 공헌이익 분석 |
| 자사몰과 오픈마켓 중 어느 채널의 수익성이 높은가? | 채널별 수수료, 광고비, 할인비 비교 |
| 홈쇼핑 의존도 감소가 고객 기반 확대로 이어지는가? | 신규 고객, 재구매 고객, 정기배송 고객 추적 |
| 온라인 고객은 반복 구매하는가? | 코호트별 재구매율과 구매 주기 분석 |

## 6. CRM 관점 분석

프롬바이오는 모바일 쇼핑 자사몰 앱을 출시했고, 공개 기사에서 정기배송과 선물하기 기능을 제공한다고 확인됩니다. 이는 자사 고객 데이터를 축적할 수 있는 접점입니다.

다만 공개 정보만으로는 앱 설치 수, 월간 활성 사용자, 고객별 구매 이력, 정기배송 전환율을 알 수 없습니다. 따라서 CRM 분석은 내부 데이터 확보 후 진행해야 합니다.

우선 분석해야 할 CRM 지표는 다음과 같습니다.

| 지표 | 분석 목적 |
| --- | --- |
| 신규 고객 수 | 온라인 채널 확장이 고객 기반 확대에 기여하는지 확인 |
| 재구매율 | 건강기능식품 제품군의 반복 구매 가능성 측정 |
| 구매 주기 | 보스웰리아, 매스틱 등 제품별 리마인드 타이밍 설계 |
| 정기배송 전환율 | 단품 구매 고객을 구독 고객으로 전환할 여지 확인 |
| 휴면 고객 비율 | 이탈 방지 캠페인 대상 규모 파악 |
| 캠페인별 구매 전환율 | CRM 메시지의 실제 매출 기여 측정 |

## 7. OEM 운영 관점 분석

2025년 보도에서는 프롬바이오의 OEM 수주 확대가 언급됩니다. 미국 건기식 업체 수주와 국내 거래처 물량 증가가 보도되었고, OEM 사업부 매출 증가와 공장 가동률 개선 가능성이 함께 언급되었습니다.

OEM 확대는 프롬바이오의 수익성 개선에 중요한 축이 될 수 있습니다. 다만 실제 효과를 판단하려면 수주 금액, 생산 원가, 납기, 공장 가동률 데이터가 필요합니다.

| 분석 질문 | 필요 데이터 |
| --- | --- |
| OEM 수주 증가가 매출과 영업이익 개선에 기여하는가? | OEM 월별 매출, 원가, 마진 |
| 공장 가동률은 개선되고 있는가? | 생산능력, 실제 생산량, 작업시간 |
| 특정 고객사 의존도가 높은가? | 고객사별 수주 비중 |
| 납기 리스크는 없는가? | 수주일, 생산일, 출고일, 납기일 |

## 9. 제안 대시보드 구성

미니 대시보드는 공개 데이터로 확인 가능한 지표와 내부 데이터가 필요한 지표를 분리해 구성합니다.

| 영역 | 공개 데이터로 구성 가능 | 내부 데이터 필요 |
| --- | --- | --- |
| 재무 | 매출, 영업손익, 분기 추세 | 제품별/채널별 손익 |
| 채널 | 온라인 비중 공개 보도 기반 전환 방향 | 월별 채널별 매출·비용 |
| CRM | 자사몰/앱 기능 존재 여부 | 재구매율, LTV, 정기배송 전환율 |
| OEM | 수주 확대 기사 기반 이벤트 타임라인 | 수주량, 가동률, 납기 준수율 |
| 실행 과제 | DX 과제 우선순위 | 과제별 ROI |

## 10. 필요한 추가 데이터

아래 데이터는 공개 자료만으로 확인되지 않았습니다. 따라서 분석 리포트의 정량 정확도를 높이려면 프롬바이오 내부 자료가 필요합니다.

| 데이터 | 없을 때 한계 |
| --- | --- |
| 월별 채널별 매출 | 온라인 전환 효과를 정확히 측정하기 어려움 |
| 채널별 비용 | 채널별 수익성, 공헌이익 계산 불가 |
| 고객별 주문 이력 | 재구매율, LTV, RFM 분석 불가 |
| 앱 설치/활성/푸시 반응 데이터 | 앱 CRM 성과 판단 불가 |
| 정기배송 가입/해지 데이터 | 구독 전환과 이탈 분석 불가 |
| OEM 수주/생산/납기 데이터 | OEM 확대의 운영 효과 판단 불가 |
| 제품별 원가/마진 | 제품 포트폴리오 개선안 산정 불가 |

## 11. 결론

공개 데이터만으로도 프롬바이오의 DX 컨설팅 방향은 비교적 명확합니다. 핵심은 온라인 채널 전환, 수익성 관리, 자사몰/앱 CRM, OEM 운영 효율화입니다.

다만 공개 데이터는 기업 전체의 방향성과 문제 신호를 파악하는 데 적합하고, 실제 ROI나 세부 개선 효과를 계산하기에는 부족합니다. 따라서 다음 단계에서는 공개 데이터를 활용한 미니 대시보드에는 확인된 재무/채널 전환 신호만 반영하고, 내부 데이터가 필요한 지표는 별도 “데이터 요청 항목”으로 표시하는 방식이 가장 안전합니다.
