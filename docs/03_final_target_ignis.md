# 1차 타깃 선정 리포트: 이그니스

## 1. 1차 선정 기업

| 항목 | 내용 |
| --- | --- |
| 기업명 | 이그니스 |
| 주요 브랜드 | 랩노쉬, 닥터랩노쉬, 한끼통살 등 |
| 산업군 | D2C 푸드/건강기능식품/라이프스타일 커머스 |
| 선정 점수 | 91점 |
| 선정 결과 | 1차 최우선 아웃바운드 DX 컨설팅 타깃 |

## 2. 선정 이유

이그니스는 건강기능식품 및 푸드 D2C 브랜드 포트폴리오를 보유하고 있으며, 멀티 브랜드 운영과 자사몰 중심 고객 관리가 중요한 기업으로 판단됩니다.

공개 채용 정보에서 CRM 마케팅, RFM 세그먼트, 타깃 캠페인 관련 신호가 확인되어 고객 데이터 기반 마케팅 고도화 니즈를 추론하기 좋습니다. 또한 브랜드와 제품군이 다양해지면서 캠페인 성과, 고객 세그먼트, 재구매율, 브랜드별 KPI를 통합적으로 관리할 필요성이 커질 가능성이 높습니다.

## 3. DX 니즈 가설

| 영역 | 현재 발생 가능 문제 | DX 컨설팅 제안 기회 |
| --- | --- | --- |
| CRM | 브랜드별 고객군과 구매 주기가 다를 가능성 | RFM 기반 고객 세그먼트 설계와 CRM 캠페인 자동화 |
| 데이터 통합 | 자사몰, 광고, 브랜드별 판매 데이터가 분산될 가능성 | 마케팅 성과 통합 대시보드 구축 |
| 리포팅 | 캠페인별 성과 보고가 수작업으로 운영될 가능성 | 주간/월간 리포트 자동화 |
| 재구매 | 제품군별 재구매 주기와 이탈 구간 관리 필요 | 리텐션 퍼널 분석과 이탈 예측 룰 설계 |
| 멀티 브랜드 운영 | 브랜드별 KPI가 따로 관리될 가능성 | 브랜드 포트폴리오 KPI 관리 체계 수립 |

## 4. 제안 가능한 DX 컨설팅 과제

| 우선순위 | 과제 | 기대 효과 |
| ---: | --- | --- |
| 1 | CRM 고객 세그먼트 진단 | 고객군별 구매 패턴, 이탈 위험군, 고가치 고객군 파악 |
| 2 | 자사몰/광고/구매 데이터 통합 대시보드 | ROAS, CAC, LTV, 재구매율을 한 화면에서 관리 |
| 3 | 캠페인 리포팅 자동화 | 반복 보고 업무 시간 절감, 의사결정 속도 개선 |
| 4 | 브랜드별 리텐션 전략 수립 | 랩노쉬, 닥터랩노쉬 등 브랜드 특성에 맞춘 CRM 운영 |
| 5 | 90일 DX 실행 로드맵 수립 | 빠른 성과 과제와 중장기 데이터 고도화 과제 분리 |

## 5. 아웃바운드 제안 메시지 방향

이그니스에는 단순한 “DX 도입”보다 “멀티 브랜드 D2C 성장에 맞는 CRM과 마케팅 데이터 운영 고도화”를 제안하는 것이 적합합니다.

제안 메시지는 다음 문제 가설을 중심으로 구성합니다.

> 브랜드와 채널이 확장될수록 CRM 캠페인, 광고 성과, 재구매 데이터를 통합적으로 보는 체계가 중요해집니다. 이그니스의 D2C 성장 단계에 맞춰 고객 세그먼트 기반 CRM 진단과 마케팅 KPI 대시보드 구축을 제안합니다.
