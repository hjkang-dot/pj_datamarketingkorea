# 후보 기업 평가 결과

## 1. 조사 대상

건강기능식품 및 D2C 커머스 성격이 있는 브랜드를 대상으로 아웃바운드 DX 컨설팅 제안 가능성을 평가했습니다.

## 3. 상위 후보 요약

### 1위: 이그니스

이그니스는 멀티 브랜드 D2C 운영, CRM 채용 신호, 고객 세그먼트 기반 캠페인 가능성이 모두 확인되어 DX 컨설팅 제안 타깃으로 가장 적합합니다. 특히 자사몰 고객 데이터를 활용한 리텐션 개선, 브랜드별 KPI 통합, CRM 자동화 제안이 자연스럽습니다.

### 2위: 뉴트리원

뉴트리원은 자사몰과 외부 판매 채널을 함께 운영하는 구조로 보이며, 채널별 성과 관리와 광고/CRM 데이터 통합 니즈를 가정하기 좋습니다. 다만 기업 규모가 상대적으로 커서 아웃바운드 제안 시 담당 조직과 의사결정 구조 확인이 필요합니다.

### 3위: 아워코퍼레이션

아워코퍼레이션은 데이터 기반 D2C 커머스 성격이 강해 성장 단계의 운영 효율화와 데이터 대시보드 제안이 적합합니다. 다만 건강기능식품 중심 포트폴리오인지, 주력 브랜드의 현재 성장 지표가 어떤지는 추가 확인이 필요합니다.
