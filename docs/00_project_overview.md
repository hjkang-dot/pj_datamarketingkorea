# 프로젝트 개요

## 1. 프로젝트 목적

본 프로젝트는 DX 컨설턴트 직무 지원용 포트폴리오로, 아웃바운드 영업 관점에서 DX 컨설팅을 제안할 만한 기업을 발굴하고 공개 데이터 기반 분석이 가능한 최종 타깃을 선정하는 과정을 다룹니다.

단순히 가상의 기업을 설정하는 방식이 아니라, 실제 시장에 존재하는 기업을 조사하고 공개 정보를 기반으로 컨설팅 니즈와 분석 가능성을 함께 판단하는 방식으로 진행합니다.

## 2. 프로젝트 주제

건강기능식품 커머스 기업 대상 공개 데이터 기반 DX 컨설팅 기회 발굴 프로젝트

## 3. 산업군 선정 이유

건강기능식품 D2C 브랜드는 DX 컨설팅 제안에 적합한 특징을 가지고 있습니다.

| 특징 | DX 컨설팅 기회 |
| --- | --- |
| 재구매와 정기구독이 중요함 | CRM 자동화, 이탈 방지, LTV 분석 |
| 광고 의존도가 높음 | ROAS, CAC, 채널별 성과 대시보드 |
| 판매 채널이 다양함 | 자사몰, 쿠팡, 스마트스토어, 올리브영 데이터 통합 |
| 리뷰와 고객 경험이 중요함 | 리뷰 분석, 고객 세그먼트 분석 |
| 브랜드 확장 속도가 빠름 | 멀티 브랜드 KPI 관리, 리포팅 자동화 |

## 4. 진행 단계

| 단계 | 내용 | 산출물 |
| --- | --- | --- |
| 1단계 | 타깃 산업군 선정 | 프로젝트 개요 |
| 2단계 | 기업 평가 기준 수립 | 타깃 기업 선정 기준표 |
| 3단계 | 후보 기업 조사 및 점수화 | 후보 기업 평가 결과 |
| 4단계 | 1차 타깃 기업 선정 | 이그니스 선정 리포트 |
| 5단계 | 공개 데이터 분석 가능성 검토 | 프롬바이오 전환 리포트 |
| 6단계 | 컨설팅 제안 방향 구체화 | 제안서, 분석 리포트, 미니 대시보드 |

## 5. 조사 방식

공개 웹 정보를 바탕으로 다음 신호를 확인했습니다.

- 자사몰 운영 여부
- 판매 채널 다양성
- 정기구독, 멤버십, 재구매 구조
- CRM, 퍼포먼스 마케팅, 데이터 분석 관련 채용 신호
- 성장성, 투자, 매출, 브랜드 확장 관련 기사
- DX 컨설팅 제안 가능성
- 공시, 재무 지표, 뉴스, 앱/자사몰 등 분석 가능한 공개 데이터

## 6. 주의 사항

본 분석은 외부 공개 정보 기반의 아웃바운드 사전 조사입니다. 실제 컨설팅 착수 전에는 기업 내부 데이터, 조직 구조, 시스템 환경, 마케팅 비용, 고객 데이터 접근 가능성에 대한 추가 확인이 필요합니다.

## 7. 타깃 전환 배경

초기에는 이그니스를 최우선 아웃바운드 타깃으로 선정했습니다. 다만 분석 리포트와 미니 대시보드까지 제작하려면 공개 데이터가 충분해야 하므로, 최종 분석 타깃은 상장사인 프롬바이오로 전환했습니다.

프롬바이오는 공시와 기사에서 매출, 영업손익, 온라인 채널 확대, 자사몰/앱, OEM 수주 확대, 유통망 개편 관련 신호가 확인되어 공개 데이터 기반 DX 컨설팅 산출물을 만들기에 더 적합합니다.
