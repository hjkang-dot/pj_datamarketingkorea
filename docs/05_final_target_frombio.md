# 최종 분석 타깃 전환 리포트: 프롬바이오

## 1. 전환 배경

초기 타깃 발굴 단계에서는 이그니스를 최우선 아웃바운드 후보로 선정했습니다. 이그니스는 D2C 브랜드 운영, CRM 채용 신호, 멀티 브랜드 구조가 뚜렷해 DX 컨설팅 제안 대상으로 매력적입니다.

다만 실제 포트폴리오 산출물로 분석 리포트와 미니 대시보드까지 만들기 위해서는 공개 데이터가 충분해야 합니다. 이그니스는 비상장 기업으로 공개 재무와 채널별 데이터가 제한적이어서 제안 내용이 가설 중심으로 흐를 가능성이 높습니다.

따라서 최종 분석 타깃은 공개 데이터 접근성이 높은 상장 건강기능식품 기업 `프롬바이오`로 전환합니다.

| 모바일 쇼핑 앱 출시 이력 | 자사몰과 모바일 고객 접점 강화 시도 | 앱 기반 CRM, 푸시 캠페인, 고객 행동 데이터 분석 |

| 건강기능식품 경쟁 심화와 홈쇼핑 채널 부진 보도 | 기존 채널 의존도 조정 필요 | 채널 포트폴리오 재설계, 마케팅 효율 분석 |

## 5. DX 니즈 가설

프롬바이오는 단순한 마케팅 CRM보다 `채널 전환 + 온라인 고객 관리 + OEM 운영 효율화`를 함께 봐야 하는 기업으로 판단됩니다.

| 영역 | 문제 가설 | 제안 방향 |
| --- | --- | --- |
| 온라인 채널 | 온라인 매출 비중은 늘고 있으나 채널별 수익성 관리가 중요할 가능성 | 자사몰, 오픈마켓, 홈쇼핑, 앱 성과 통합 대시보드 |
| CRM | 자사몰/앱 고객의 재구매와 정기배송 전환 관리 필요 | 구매 주기 기반 CRM 캠페인 자동화 |
| 수익성 | 매출 회복보다 영업손익 개선이 핵심 과제일 가능성 | 채널별 공헌이익, 광고비, 할인비, 물류비 관리 |
| OEM | OEM 수주 증가가 공장 가동률과 원가율 개선에 연결될 가능성 | OEM 수주/생산/납기 KPI 모니터링 |
| 리포팅 | 유통망 개편 과정에서 경영진 보고 지표가 복잡해질 가능성 | 월간 경영관리 리포트 자동화 |

## 6. 제안 가능한 DX 컨설팅 과제

| 우선순위 | 과제 | 기대 효과 |
| ---: | --- | --- |
| 1 | 채널별 매출·수익성 통합 대시보드 | 홈쇼핑, 자사몰, 오픈마켓, OEM 성과를 같은 기준으로 비교 |
| 2 | 자사몰/앱 CRM 진단 | 온라인 고객 재구매율, 구매 주기, 이탈 위험군 파악 |
| 3 | 온라인 채널 전환 성과 분석 | 유통망 개편이 매출과 수익성에 미치는 효과 측정 |
| 4 | OEM 수주·생산 KPI 관리 | 수주 증가와 공장 가동률, 원가율 개선의 연결 구조 확인 |
| 5 | 경영진 월간 리포트 자동화 | 매출, 손익, 채널 비중, 재구매 지표를 반복 보고서로 자동화 |
