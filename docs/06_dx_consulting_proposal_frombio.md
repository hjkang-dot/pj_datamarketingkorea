# DX 컨설팅 제안서: 프롬바이오

## 1. 제안 개요

본 제안서는 프롬바이오의 수익성 개선과 채널 전환을 지원하기 위한 DX 컨설팅 제안입니다.

공개 자료 기준으로 프롬바이오는 건강기능식품 제조 및 판매를 주력으로 하며, 2023년 연결 기준 매출 감소와 영업손실을 경험했습니다. 이후 온라인 채널 비중 확대, 자사몰/모바일 앱, OEM 수주 확대, 생산능력 활용과 원가 개선이 중요한 과제로 나타납니다.

따라서 본 컨설팅은 `채널별 수익성 관리`, `온라인 고객 CRM`, `OEM 수주·생산 KPI`, `경영진 리포팅 자동화`를 중심으로 설계합니다.

## 3. 핵심 문제 정의

프롬바이오의 DX 과제는 단순히 온라인몰을 고도화하는 것이 아니라, 채널 구조 변화와 수익성 개선을 동시에 관리하는 것입니다.

| 문제 영역 | 문제 정의 | DX 관점 |
| --- | --- | --- |
| 채널 전환 | 홈쇼핑 중심 구조에서 온라인·오프라인·OEM으로 채널이 재편되는 상황 | 채널별 매출, 비용, 공헌이익을 통합 관리 |
| 수익성 | 매출 회복만으로는 부족하고 비용 구조 개선이 필요 | 광고비, 수수료, 할인, 물류비를 KPI화 |
| 온라인 CRM | 자사몰/앱 비중이 커질수록 고객 재구매 관리가 중요 | 구매 주기, 재구매율, 이탈 위험군 분석 |
| OEM 운영 | OEM 수주 확대가 생산 가동률과 원가율 개선에 연결되어야 함 | 수주, 생산, 납기, 가동률 대시보드 |
| 보고 체계 | 채널과 사업 모델이 늘어나면 경영진 보고 지표가 복잡해짐 | 월간 경영관리 리포트 자동화 |

| 온라인 고객 관리 강화 | 자사몰/앱 고객의 재구매, 정기배송, 이탈 위험을 분석 |

| 90일 실행 로드맵 수립 | 빠른 성과 과제와 내부 데이터 정비 과제를 분리 |

## 5. 제안 과제

### 과제 1. 채널별 매출·수익성 통합 대시보드

프롬바이오의 채널 전환 상황을 관리하기 위해 매출액뿐 아니라 채널별 비용과 공헌이익을 함께 보는 대시보드를 설계합니다.

| KPI | 설명 | 필요 데이터 |
| --- | --- | --- |
| 채널별 매출 | 홈쇼핑, 자사몰, 오픈마켓, 오프라인, OEM 매출 | 주문/매출 데이터 |
| 채널별 비용 | 판매수수료, 광고비, 할인비, 물류비 | 회계/마케팅 비용 데이터 |
| 공헌이익 | 채널 매출에서 직접 비용을 차감한 수익성 | 매출원가, 변동비 |
| 온라인 매출 비중 | 전체 매출 중 자사몰/앱/오픈마켓 비중 | 채널 구분 매출 |
| 월별 추세 | 채널 전환이 손익에 주는 영향 | 월별 손익 데이터 |

### 과제 2. 자사몰·앱 CRM 진단

모바일 앱과 자사몰 고객 데이터를 기반으로 재구매와 정기배송 전환을 관리합니다.

| 분석 항목 | 활용 방안 | 필요 데이터 |
| --- | --- | --- |
| RFM 세그먼트 | 고가치 고객, 신규 고객, 휴면 위험 고객 분류 | 고객별 주문 이력 |
| 구매 주기 | 제품별 재구매 리마인드 시점 설계 | 상품별 구매 간격 |
| 정기배송 전환 | 단품 구매 고객의 구독 전환 후보 발굴 | 정기배송 여부 |
| 앱 캠페인 성과 | 푸시/쿠폰/이벤트별 구매 전환 측정 | 앱 이벤트, 캠페인 로그 |

### 과제 3. OEM 수주·생산 KPI 관리

OEM 수주 확대가 실제 수익성 개선으로 이어지는지 확인하기 위해 수주와 생산 운영 지표를 연결합니다.

| KPI | 설명 | 필요 데이터 |
| --- | --- | --- |
| 수주 물량 | 고객사별 월별 수주량 | OEM 계약/수주 데이터 |
| 생산 계획 대비 실적 | 계획 물량 대비 실제 생산량 | 생산 실적 데이터 |
| 공장 가동률 | 설비 능력 대비 실제 생산량 | 생산능력, 작업시간 |
| 납기 준수율 | 약속 납기 대비 출고 완료율 | 출고/납기 데이터 |
| 제품별 원가율 | OEM 제품별 원가 구조 | 원가 데이터 |

### 과제 4. 경영진 월간 리포트 자동화

매월 반복되는 실적 취합과 보고서 작성을 자동화하기 위해 핵심 지표와 템플릿을 표준화합니다.

| 리포트 영역 | 포함 지표 |
| --- | --- |
| 손익 | 매출, 영업손익, 매출총이익률, 판관비 |
| 채널 | 채널별 매출, 온라인 비중, 홈쇼핑 의존도 |
| 고객 | 신규 고객, 재구매율, 정기배송 전환율 |
| OEM | 수주량, 생산량, 가동률, 납기 준수율 |
| 실행 과제 | 비용 절감 과제, CRM 캠페인, 채널 개선 액션 |

## 10. 제안 결론

프롬바이오는 온라인 채널 전환, 자사몰/앱 고객 관리, OEM 수주 확대, 수익성 개선이라는 여러 과제가 동시에 존재합니다. 이 과제들은 각각 따로 관리하면 성과를 판단하기 어렵기 때문에, 채널별 손익과 고객 CRM, OEM 운영 KPI를 하나의 경영관리 체계로 묶는 DX 컨설팅이 필요합니다.

본 프로젝트는 대규모 시스템 구축보다 4주 진단과 90일 실행 로드맵을 통해 핵심 KPI, 데이터 요구사항, 대시보드 프로토타입을 빠르게 정리하는 방식으로 추진하는 것을 제안합니다.
